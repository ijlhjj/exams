# 杨辉三角 II

## 题目描述

给定一个非负索引 rowIndex，返回「杨辉三角」的第 rowIndex 行。

在「杨辉三角」中，每个数是它左上方和右上方的数的和。

### 示例 1

```text
    输入: rowIndex = 3
    输出: [1,3,3,1]
```

### 示例 2

```text
    输入: rowIndex = 0
    输出: [1]
```

### 示例 3

```text
    输入: rowIndex = 1
    输出: [1,1]
```

### 提示

```text
    * 0 <= rowIndex <= 33
```

### 进阶

你可以优化你的算法到 O(rowIndex) 空间复杂度吗？
